# 战利品袋

## 🪄介绍

又一个战利品袋模组，为魔改而生。

![image](assets/loot-bag/loot-bag.png)

右键打开一个预览屏幕。
潜行右键直接打开战利品袋。

### (单项) 战利品袋

包含单组战利品。

![image](assets/loot-bag/single.png)

### 可选战利品袋

包含多组战利品，你可以选择获取其中一组。

![image](assets/loot-bag/optional.gif)

### 随机战利品袋

包含多组战利品，你可以获取其中随机一组。

![image](assets/loot-bag/random.gif)
